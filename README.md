# Nafa AI - Manual Testing 

This repository contains the manual testing documentation, bug reports, and the latest stable build for **Nafa AI**, a personalized investment guidance platform.

## 🚀 Project Overview
I have conducted an end-to-end manual SQA (Software Quality Assurance) audit of the Nafa AI mobile application. This includes testing the user journey from onboarding and risk assessment to investment recommendations and learning modules.
